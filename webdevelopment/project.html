Projects and Experience

This page highlights professional experience and academic projects that have helped me grow in information technology. The examples focus on system support, information access, networking, programming, and database work.

Professional Projects

These projects show how I use technology to support employees and improve access to important information.

Records Management System Experience

I contributed to work involving an in-house records management system developed with an application development team. The system supports operational needs by organizing records and giving authorized employees access to accurate information.

Reflection

This experience helped me understand that successful technology projects must solve real user problems. Reliability, security, and clear workflows are all important when employees depend on a system during daily operations.

Internal Dashboard Development

I created intranet-accessible dashboards that provide updates on personnel workloads, server workloads, and instructions for accessing internal systems. The goal was to make frequently requested information easier for both technical and nontechnical employees to find.

Reflection

This project showed me how useful a straightforward dashboard can be when it is designed around the needs of its users. Clear labels and current information can reduce confusion and save employees time.

Technical Projects

These projects support my interests in network engineering, security, and infrastructure troubleshooting.

Virtual Network Labs

I use virtual network labs to practice routing, switching, IP addressing, connectivity testing, and troubleshooting. A controlled lab provides a safe place to test configurations and better understand how devices communicate across different networks.

Reflection

Virtual labs help connect classroom concepts to actual troubleshooting steps. They allow me to practice commands, make configuration changes, identify mistakes, and verify connectivity without affecting a production network.

Programming and Database Coursework

My academic projects have included Python applications, SQL queries, database design, and web development. These assignments gave me experience organizing data, writing logic, testing results, and correcting problems before considering a project complete.

Reflection
